Historia dos computadores

(volta)

O computador é uma maquina programada capaz de resolver problemas complexos auxiliando nas atividades humanas.

Muito antes dos computadores eletrônicos, equipamentos com a mesma finalidade foram criados, como por exemplo:

Abaco, Wilhelm Schickard, Blaise Pascal, Cartão perfurado, Hollerith, entre outros.

Teoria da computabilidade

O inglês Alan Turing com objetivo em decifrar os códigos alemãs durante os períodos da guerra, criou a maquina de turing com formalismo matemático para criação de algoritmos, com isso a ideia do computador foi plantada.

Colossus

O Thomas Flowers (inglês)-1942 criador do primeiro computador eletrônico chamado Colossus, utilizando a mesma ideia de Alan Turing com objetivo de analise de dados, controle de fluxo, Simulação de máquinas e o principal objetivo a decifração de códigos.

  1. Arquitetura: computador eletrônico digital
  2. Memoria 2000 bits de memória de estado sólido
  3. Velocidade: 25 kHz (100.000 operações por segundo aproximadamente)
  4. tamanho de 7x2 metros
  5. Peso de 1 tonelada

Arquitetura binaria

Von Neumann - 1945 definiu a Arquitetura computacional utilizada até os dias de hoje, criou o computador binário 0 e 1.

“O programa que direciona as atividades da CPU é armazenado na mesma memória em que estão os dados, que devem ser manipulados pelo programa”

o computador é uma máquina de programas armazenados.

ENIAC

Desenvolvido durante a segunda guerra, seu principal objetivo era calcular tabelas de tiros para o exercito dos Estados Unidos.

  1. Arquitetura: computador eletrônico digital
  2. Computador multipropósito
  3. Ocupava 180 m² de área construída
  4. 70 mil resistores
  5. Ocupava uma sala de no primeiro andar da Moore School, oeste da Filadélfia.

Primeiro PC - 1981

Lançado 12 de agosto de 1981, o primeiro computador pessoal, possia uma Memória de 16 Kbytes, Processador Intel de 4,77 MHz e custava US$ 1.565.

O que são Clients?

(volta)

Client é um termo empregado em computação e representa uma entidade que consome os serviços de uma outra entidade servidora, em geral através do uso de uma rede de computadores numa arquitetura cliente-servidor.

O que são Serves?

(volta)

um servidor é um software ou computador, com sistema de computação centralizada que fornece serviços a uma rede de computadores, chamada de cliente.

Esta arquitetura é chamada de modelo cliente-servidor, é utilizada em redes de médio e grande porte (com muitas máquinas) e em redes onde a questão da segurança desempenha um papel de grande importância. O termo servidor é amplamente aplicado a computadores completos, embora um servidor possa equivaler a um software ou a partes de um sistema computacional, ou até mesmo a uma máquina que não seja necessariamente um computador.

Os servidores podem fornecer várias funcionalidades, muitas vezes chamado de "serviços", tais como a partilha de dados ou de recursos do sistema entre vários clientes, ou computação desempenho para um cliente. Um único servidor pode servir vários clientes, e um único cliente pode usar vários servidores. Um processo cliente pode ser executado no mesmo dispositivo ou pode se conectar através de uma rede para um servidor em um dispositivo diferente.

Estrutura básica de um HTML

(volta)

Um documento HTML valido precisa obrigatoriamente de uma estrutura composta pelas seguintes tags:

  • tag Html - Define um documento HTML
  • tag Head - Contem algumas informações que não são visíveis aos usuários, como as metas tags, tags links, entre outros.
  • tag body - Tags onde vão os elementos da pagina visíveis ao usuário, como header, main, footer, etc.

Listas ordenadas e não ordenadas

(volta)

Lista não ordenadas necessariamente não precisam de uma ordem especifica, e são definidas pelas tags ul (lista não ordenada) li (item da lista).

  • Item 1
  • Item 6
  • Item 4

Lista ordenadas precisam de uma ordem especifica como um passo a passo, e são definidas pelas tags ol (lista ordenada) li (item da lista).

  1. Primeiro processo
  2. Segundo processo
  3. Terceiro processo
